📚 Documentación API WhatsApp

API REST para enviar y recibir mensajes de WhatsApp usando el protocolo WAHA (WhatsApp HTTP API).

🔗 Información General

  • URL Base: https://api.expertosenia.com
  • Autenticación: Header ex-ia-api
  • Formato: JSON
  • Protocolo: HTTPS

🔐 Autenticación

Todas las peticiones requieren el header de autenticación:

Headers: Content-Type: application/json ex-ia-api: TU_API_KEY_AQUI

📱 Sesiones

La API maneja sesiones de WhatsApp. Si no especificas una sesión, se usa "default".

{ "session": "default", // ... otros parámetros }
